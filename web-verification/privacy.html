Politique de Confidentialité

1. Collecte d'Informations

Nous collectons uniquement les informations nécessaires pour la vérification Discord :

  • ID utilisateur Discord
  • Nom d'utilisateur Discord
  • Avatar Discord
  • Rôles sur le serveur

2. Utilisation des Données

Les données collectées sont utilisées exclusivement pour :

  • Vérifier votre identité Discord
  • Attribuer les rôles appropriés
  • Assurer la sécurité du serveur

3. Stockage des Données

Important : Aucune donnée personnelle n'est stockée de manière permanente. Les informations sont traitées en temps réel et supprimées immédiatement après utilisation.

4. Partage des Données

Nous ne partageons, ne vendons, ni ne transmettons vos données personnelles à des tiers. Les données sont utilisées uniquement par notre système de vérification.

5. Sécurité

Nous utilisons des protocoles de sécurité standard pour protéger vos données :

  • HTTPS pour toutes les communications
  • OAuth2 Discord pour l'authentification
  • Aucun stockage local de données sensibles

6. Cookies

Nous n'utilisons pas de cookies de tracking. Seuls les cookies techniques nécessaires au fonctionnement du service peuvent être utilisés.

7. Vos Droits

Conformément au RGPD, vous avez le droit de :

  • Demander l'accès à vos données
  • Demander la suppression de vos données
  • Retirer votre consentement à tout moment

8. Contact

Pour toute question concernant cette politique de confidentialité, contactez-nous sur Discord.
